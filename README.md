# tasuke 助

Free, member-authored cybercrime-victim recovery support: intake, triage, evidence
preservation, document drafting, free public-window routing, and account recovery.
It never adjudicates a crime and never submits on behalf of a member.

Canonical ontology: `schema/cybercrime-victim-support-ontology.kotoba.edn`.
Canonical seed: `data/seed-cybercrime-cases.kotoba.edn`.
Run the portable suite with `bb run_tests.clj`.
